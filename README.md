# Relay

Relay is a tiny macOS menu-bar app that syncs your iMessage history to your private workspace.

## Download

**[Download the latest version](https://github.com/lucacharrouf/relay-releases/releases/latest)**

### Install
1. Open the downloaded `.dmg`
2. Drag **Relay** to your **Applications** folder
3. Launch Relay from Applications (or Spotlight)
4. Grant **Full Disk Access** when prompted — this is required to read `~/Library/Messages/chat.db`

## Requirements
- macOS 13 (Ventura) or later
- Apple Silicon or Intel

## Privacy
Relay reads your local iMessage database and uploads message metadata to your account. It does not read or transmit attachments. All connections are HTTPS.
